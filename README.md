# [WIP]litJSX

A 10kb only JSX compiler running in browser"

# Usage: 
```js        
litJSX.jsx(React, {Tag: Tag})`
    <Tag ...${{test:"included"}}  v${"a"}l=t${"es"}t val1=t${"es"}t1 v${"a"}l2=test2/>
`
```
